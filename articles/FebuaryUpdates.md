# Febuary Updates
---

WRITTEN BY JACK

EDITED BY EULM

***

This February, many things have changed in BendersMC, some of which have smaller and greater impacts. This includes the new seasonal sets, the Valentine set and the Eternal Frost set, the new main lobby, and the new chat colour format.

##### Seasonal Sets
The new seasonal sets, the Valentine set and the Eternal Frost set, are especially interesting in a few ways. First of all, the Valentine set has one thing that we didn't see in the original version last year: it has an elytra. This is the first seasonal set including an elytra for half a year and it is super cool to have this feature back. Secondly, the Eternal Frost set has traits seen before but many claim it is drastically different from the Winter set and similar others. It is in a familiar winter style and it comes with a full set of tools, armour and cosmetics.

##### New Main Lobby
One thing that is extremely new is the main lobby. After keeping it unchanged since 2024, it's finally been revamped, and it's now more open and is overall a brilliant build that the build team has worked on a lot. This new lobby is a more stone textured, mythical lobby that looks amazing and suits the BendersMC aesthetic.

##### Chat Colour 
A lot of people say that the chat colour is something that makes businesses on BendersMC thrive and sustain wealth and prosperity on the server. Recently, the chat color format has been reversed back to the format that you can generate with RGBirdflop. It is a simple format but unlike the previous one, it takes up a lot of space. This was the original format in BendersMC, familiar to many businesses.

##### Conclusion
The chat colour, new seasonal sets, and new lobby bring about new additions and change on BendersMC with a lot of mixed reviews. One thing is certain though: they change the future of BendersMC in a way that shapes businesses, clans and the player experience for the foreseeable future on the server.
